Grab and modify notes with our Note Taker Api

To see all note JSON do a GET request here -> /api/notes

To see a individual note JSON do a GET request /api/notes/:id (enter the note id of the note you wish to grab the json)

To create a new individual note do a POST request /api/notes

To modify a individual note JSON do a PUT request here ->/api/notes/:id (enter the note id of the note you wish to grab the json)
